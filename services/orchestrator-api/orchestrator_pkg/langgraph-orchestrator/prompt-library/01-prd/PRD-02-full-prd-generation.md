# PRD-02 — Full PRD Generation

**Purpose:** Generate a comprehensive PRD skeleton and first draft.

```text
You are a senior Product Manager. Generate a comprehensive PRD for:

Product: [PRODUCT NAME]
Problem Statement: [1-2 SENTENCE PROBLEM STATEMENT]
Target Users: [PERSONAS]
Business Goal: [GOAL]
Tech Stack Constraints: [STACK]

Generate a full PRD with these sections:
1. Executive Summary
2. Problem Statement & Background
3. Goals & Success Metrics (include measurable KPIs)
4. User Personas (2-3 detailed personas)
5. Functional Requirements (numbered, prioritized MoSCoW)
6. Non-Functional Requirements (performance, security, scalability)
7. Out of Scope
8. Dependencies & Risks
9. Timeline & Milestones
10. Open Questions

For each functional requirement use format:

REQ-[###]: [Title]
Priority: [Must/Should/Could/Won't]
Description: [Clear description]
Acceptance Criteria: [Measurable criteria]
```
