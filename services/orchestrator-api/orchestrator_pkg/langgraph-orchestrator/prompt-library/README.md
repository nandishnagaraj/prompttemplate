# GenAI & Agentic AI — Prompt Library (Markdown)

This folder contains reusable prompt templates organized by SDLC stage, plus agentic guardrails and best-practice patterns.

## Contents

- `00-core/` — prompting frameworks and reusable patterns
- `01-prd/` — PRD and requirements prompts
- `02-user-stories/` — user stories, acceptance criteria, dependency mapping
- `03-design/` — architecture, data model, API contracts, risk review
- `04-development/` — implementation, review, refactor, documentation
- `05-testing/` — unit/integration tests, coverage gaps, security testing
- `06-release/` — release readiness, notes, rollout, observability
- `07-agentic/` — agent system prompt template + guardrails
