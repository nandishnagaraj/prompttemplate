# User Story Quality Framework (INVEST)

Use AI to validate each story against INVEST.

| Criterion | What to Verify |
|---|---|
| Independent | Can be developed without blocking others |
| Negotiable | Details flexible until committed |
| Valuable | Delivers user/business value |
| Estimable | Team can size confidently |
| Small | Completes within a sprint |
| Testable | Concrete, verifiable acceptance criteria |
