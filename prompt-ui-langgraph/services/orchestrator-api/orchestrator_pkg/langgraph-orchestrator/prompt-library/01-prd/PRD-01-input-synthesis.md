# PRD-01 — PRD Input Synthesis

**Purpose:** Synthesize raw stakeholder notes + research into an AI-ready PRD starter.

```text
You are a senior Product Manager creating a PRD for a software product.

Context:
- Product: [PRODUCT NAME]
- Company: [COMPANY TYPE / DOMAIN]
- Target Users: [USER PERSONAS]

Raw inputs collected:
<stakeholder_notes>
[PASTE NOTES HERE]
</stakeholder_notes>

<market_research>
[PASTE RESEARCH HERE]
</market_research>

Your task:
1. Identify the core problem being solved
2. Extract 5-8 key business objectives
3. List assumptions and open questions
4. Suggest success metrics (KPIs)

Output as structured markdown with clear sections.
```
