# DES-01 — High-Level Architecture Design

**Purpose:** Produce component architecture, flows, ADRs, scaling + topology.

```text
You are a senior software architect.

Requirements Summary:
[PASTE KEY NFRs AND FUNCTIONAL REQUIREMENTS]

Tech Stack:
- Frontend: [STACK]
- Backend: [STACK]
- Database: [DB]
- Infrastructure: [CLOUD/ON-PREM]

Design a high-level architecture covering:
1. System components and their responsibilities
2. Component interaction diagram (text-based, using ASCII or mermaid.js)
3. Data flow for the 3 most critical user journeys
4. Technology choices with justification
5. Scalability strategy (horizontal/vertical scaling approach)
6. Single points of failure and mitigation strategies
7. Deployment topology (environments: dev/staging/prod)

For each architectural decision, document using ADR format:
Title | Context | Decision | Consequences
```
