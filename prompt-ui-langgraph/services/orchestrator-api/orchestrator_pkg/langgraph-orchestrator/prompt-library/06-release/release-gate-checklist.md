# Release Gate Checklist

- [ ] All P0/P1 user stories completed and verified
- [ ] Zero Critical / High-severity bugs open
- [ ] Test coverage >= defined threshold (e.g., 80%)
- [ ] Security scan: no new Critical/High vulnerabilities
- [ ] Performance tests: all SLOs met under projected load
- [ ] Feature flags configured and tested
- [ ] Rollback procedure tested in staging
- [ ] Monitoring dashboards and alerts active
- [ ] Runbooks updated and reviewed
- [ ] Product Owner and stakeholders have signed off
- [ ] Communication to users/customers ready
