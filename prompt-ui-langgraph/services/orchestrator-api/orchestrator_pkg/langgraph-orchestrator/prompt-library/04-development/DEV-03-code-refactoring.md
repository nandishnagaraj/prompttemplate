# DEV-03 — Code Refactoring

**Purpose:** Targeted refactoring with behavior preserved and minimal changes.

```text
Refactor the following code to improve [SPECIFIC GOAL: readability / performance / testability].

<original_code>
[PASTE CODE]
</original_code>

Refactoring goals:
1. [SPECIFIC GOAL 1 — e.g., extract functions > 30 lines]
2. [SPECIFIC GOAL 2 — e.g., replace nested conditionals with early returns]
3. [SPECIFIC GOAL 3 — e.g., remove duplication]

Constraints:
- Preserve all existing behavior (do not change external interface)
- Maintain same function/method signatures
- Keep changes minimal — do not add new features

Explain each change you make and why.
```
