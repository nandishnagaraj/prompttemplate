# Definition of Done (Template)

- Code written and peer-reviewed via pull request
- Unit tests written with >= 80% code coverage
- Acceptance criteria verified by QA (manual or automated)
- API documentation updated (Swagger/OpenAPI)
- No critical or high-severity static analysis issues
- Performance benchmarks met as defined in NFRs
- Security scan completed with no new critical findings
- Feature flag configured (if applicable)
- Product Owner sign-off on acceptance criteria
- Deployed to staging successfully
