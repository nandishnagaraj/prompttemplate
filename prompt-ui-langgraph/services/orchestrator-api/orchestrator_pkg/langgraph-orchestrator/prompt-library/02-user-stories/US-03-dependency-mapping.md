# US-03 — Story Dependency & Sequencing

**Purpose:** Identify dependencies, propose sprint plan, define MVP slice, flag risks.

```text
Given these user stories:

<stories>
[PASTE STORY LIST]
</stories>

Analyze and produce:
1. Dependency graph (which stories must be done before others)
2. Suggested sprint breakdown (2-week sprints)
3. MVP slice (minimum stories to deliver core value)
4. Risk flags (stories with unclear dependencies or high complexity)

Format the sprint plan as a table with:
Sprint | Story ID | Story Title | Points | Dependencies
```
